LIGHTSTREAMER - NODE.JS CLIENTS (UNIFIED API) - USAGE INSTRUCTIONS
==================================================================

The library is deployed on the npm repository at https://www.npmjs.com/package/lightstreamer-client

To include the library in a custom project it is necessary to configure
the dependency on the project's package.json.

Example
=======

{
  "name": "my-app",
  "version": "1.0.0",
  .....
  "dependencies": {
    "lightstreamer-client": ">=7.3.0"
  }
}
