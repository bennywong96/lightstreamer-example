LIGHTSTREAMER - NODE.JS ADAPTERS - USAGE INSTRUCTIONS
=====================================================

The library is deployed on the npm repository at https://www.npmjs.com/package/lightstreamer-adapter

To include the library in a custom project it is necessary to configure
the dependency on the project's package.json.

Example
=======

{
  "name": "my-app",
  "version": "1.0.0",
  .....
  "dependencies": {
    "lightstreamer-adapter": ">=1.4.2"
  }
}
