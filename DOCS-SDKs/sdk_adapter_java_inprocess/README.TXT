LIGHTSTREAMER - JAVA ADAPTER DEVELOPMENT RESOURCES
==================================================

This directory includes the resources needed to develop Data Adapters and
Metadata Adapters that are pluggable into Lightstreamer Server.

- Version.txt
  Reports the current SDK version and compatibility constraints.

- doc/API-reference/*
  The Adapter API Interface class javadoc files.

- doc/API-reference-index.html
  The home page for the Adapter API Interface class javadocs.
  
- doc/adapter_conf_template/adapters.xml
  A generic template for the configuration file of an Adapter Set
  pluggable into Lightstreamer Server.

- A "Hello World" tutorial that covers the development of a simple Java
  Data Adapter can be found on GitHub:
  https://github.com/Lightstreamer/Lightstreamer-example-HelloWorld-adapter-java.

- lib/ls-adapter-interface.jar
  Contains the interfaces to be implemented by Data Adapters and Metadata
  Adapters, to be used in order to compile the Adapters.

- examples/examples.html
- examples/examples.txt
  Links to online examples for this SDK.
