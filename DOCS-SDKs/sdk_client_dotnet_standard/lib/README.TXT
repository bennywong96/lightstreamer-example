LIGHTSTREAMER - .NET Standard CLIENTS - USAGE INSTRUCTIONS
==========================================================

The library is deployed through the NuGet package manager (nuget.org). 
The library can be downloaded directly from: https://www.nuget.org/packages/Lightstreamer.DotNetStandard.Client/

Alternatively to include the library in a custom VS project follow these steps:

1. In the "Solution Explorer" tab, right click on the project and choose "Manage NuGet Packages ..."
2. In the Search text box enter "Lightstreamer"
3. Choose Lightstreamer.DotNetStandard.Client then click "Install" and then "Ok"
4. Check out that among the References of your project Lightstreamer_DotNet_Standard_Client was added.
