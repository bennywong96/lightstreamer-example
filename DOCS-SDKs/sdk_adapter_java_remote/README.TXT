LIGHTSTREAMER - JAVA REMOTE ADAPTER DEVELOPMENT RESOURCES
=========================================================

This directory includes the resources needed to develop Remote Data Adapters and
Remote Metadata Adapters in Java. Java 7 or higher is supported.
The adapters will run in a separate process, communicating with the Server
through the Proxy Adapters.

Refer to the Lightstreamer/DOCS-SDKs/adapter_remoting_infrastructure for
an overview of the general architecture and deploy instructions for the Proxy
Adapters.

- Version.txt
  Reports the current SDK version and compatibility constraints.

- doc/API-reference/*
  The Adapter API Interface class javadoc files.

- doc/API-reference-index.html
  The home page for the Adapter API Interface class javadocs.

- lib/ls-adapter-remote.jar
  Contains the interfaces to be implemented by Data Adapters and Metadata
  Adapters, and the classes to be used to run the Remote Adapters in a 
  Java Virtual Machine.
  
- examples/examples.html
- examples/examples.txt
  Links to online examples for this SDK.
