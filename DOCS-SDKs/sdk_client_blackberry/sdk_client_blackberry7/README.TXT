LIGHTSTREAMER - BLACKBERRY OS CLIENT DEVELOPMENT RESOURCES
==========================================================

This directory includes the resources needed to develop a BlackBerry 
application that works as a client of Lightstreamer Server.

This SDK is designed for BlackBerry up to version 7.

- Version.txt
  Reports the current SDK version and compatibility constraints.

- doc/API-reference/*
  The Client API Interface class javadoc files.

- doc/API-reference-index.html
  The home page for the Client API Interface class javadocs.

- lib/ls-bb-client.jar
  Client library to be included in BlackBerry applications.
  You may unjar this file in order to pack its classes with your own
  in a new jar.

- examples/examples.html
- examples/examples.txt
  Links to online examples for this SDK.
