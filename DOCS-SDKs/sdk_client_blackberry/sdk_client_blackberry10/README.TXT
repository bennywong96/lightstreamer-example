LIGHTSTREAMER - BLACKBERRY 10 CLIENT DEVELOPMENT RESOURCES
==========================================================

This directory includes the resources needed to make a BlackBerry 10 application a Client 
of Lightstreamer Server, in order to get real-time streaming data.

There is no dedicated library for the BlackBerry 10 OS, instead the Flex, Web (Unified API),
Android, and Android (Unified API) Client Libraries can be exploited to prepare applications
to be run on a BlackBerry 10 device. 
In the examples folder you can find the links to 3 simple applications, one per technology.

- examples/examples.html
- examples/examples.txt
  Links to online examples for this SDK.
